NamedExec

  • DB.NamedExec:方法用来绑定SQL语句与结构体或map中的同名字段,使用了KV形式

NamedQuery

  • NamedQuery:KV下的查询方法

事务操作

  • db.Beginx(): 开启事务
  • tx.Exec(): 执行事务
  • tx.Commit(): 提交事务
  • tx.Rollback(): 事务回滚
