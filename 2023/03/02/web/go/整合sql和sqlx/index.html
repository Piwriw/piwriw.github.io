What is sqlx

  • sqlx是Go语言内置database/sql的超集，它在优秀的内置database/sql基础上提供了一组扩展

进阶特性用法

NamedExec

  • DB.NamedExec:方法用来绑定SQL语句与结构体或map中的同名字段,使用了KV形式

NamedQuery

  • NamedQuery:KV下的查询方法

事务操作

  • db.Beginx(): 开启事务
  • tx.Exec(): 执行事务
  • tx.Commit(): 提交事务
  • tx.Rollback(): 事务回滚

sqlx.In批量插入

  • 其中sqlx.in注意使用...
  • 关于sqlx.In一个用法，在sqlx查询语句中实现In查询和FIND_IN_SET函数。即实现SELECT * FROM user WHERE id in (3, 2, 1);和SELECT * FROM user WHERE id in (3, 2, 1) ORDER BY FIND_IN_SET(id, ‘3,2,1’)
