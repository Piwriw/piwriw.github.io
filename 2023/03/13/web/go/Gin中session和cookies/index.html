Ginдёӯsessionе’Ңcookies

session

д»Җд№ҲжҳҜsession

  • SessionжҳҜеңЁж— зҠ¶жҖҒзҡ„HTTPеҚҸи®®дёӢпјҢжңҚеҠЎз«Ҝи®°еҪ•з”ЁжҲ·зҠ¶жҖҒж—¶з”ЁдәҺж ҮиҜҶе…·дҪ“з”ЁжҲ·зҡ„жңәеҲ¶
  • е®ғжҳҜеңЁжңҚеҠЎз«Ҝдҝқеӯҳзҡ„з”ЁжқҘи·ҹиёӘз”ЁжҲ·зҡ„зҠ¶жҖҒзҡ„ж•°жҚ®з»“жһ„пјҢеҸҜд»ҘдҝқеӯҳеңЁж–Үд»¶гҖҒж•°жҚ®еә“жҲ–иҖ…йӣҶзҫӨдёӯ
  • еңЁжөҸи§ҲеҷЁе…ій—ӯеҗҺиҝҷж¬Ўзҡ„Sessionе°ұж¶ҲеӨұдәҶпјҢдёӢж¬Ўжү“ејҖе°ұдёҚеҶҚжӢҘжңүиҝҷдёӘSessionгҖӮе…¶е®һе№¶дёҚжҳҜSessionж¶ҲеӨұдәҶпјҢиҖҢжҳҜSession IDеҸҳдәҶ

cookie

д»Җд№ҲжҳҜcookieпјҹ

  • CookieжҳҜе®ўжҲ·з«Ҝдҝқеӯҳз”ЁжҲ·дҝЎжҒҜзҡ„дёҖз§ҚжңәеҲ¶пјҢз”ЁжқҘи®°еҪ•з”ЁжҲ·зҡ„дёҖдәӣдҝЎжҒҜ
  • жҜҸж¬ЎHTTPиҜ·жұӮж—¶пјҢе®ўжҲ·з«ҜйғҪдјҡеҸ‘йҖҒзӣёеә”зҡ„CookieдҝЎжҒҜеҲ°жңҚеҠЎз«ҜгҖӮе®ғзҡ„иҝҮжңҹж—¶й—ҙеҸҜд»Ҙд»»ж„Ҹи®ҫзҪ®пјҢеҰӮжһңдҪ дёҚдё»еҠЁжё…йҷӨе®ғпјҢдҝқз•ҷ

дәҢиҖ…еҜ№жҜ”

  1. Cookie еңЁе®ўжҲ·з«ҜпјҲжөҸи§ҲеҷЁпјүпјҢSession еңЁжңҚеҠЎеҷЁз«Ҝ
  2. Cookieзҡ„е®үе…ЁжҖ§дёҖиҲ¬пјҢд»–дәәеҸҜйҖҡиҝҮеҲҶжһҗеӯҳж”ҫеңЁжң¬ең°зҡ„Cookieе№¶иҝӣиЎҢCookieж¬әйӘ—гҖӮеңЁе®үе…ЁжҖ§з¬¬дёҖзҡ„еүҚжҸҗдёӢпјҢйҖүжӢ©SessionжӣҙдјҳгҖӮйҮҚиҰҒдәӨдә’дҝЎжҒҜжҜ”еҰӮжқғйҷҗзӯүе°ұиҰҒж”ҫеңЁSessionдёӯпјҢдёҖиҲ¬зҡ„дҝЎжҒҜи®°еҪ•ж”ҫCookieе°ұеҘҪдәҶ
  3. еҚ•дёӘCookieдҝқеӯҳзҡ„ж•°жҚ®дёҚиғҪи¶…иҝҮ4KпјҢеҫҲеӨҡжөҸи§ҲеҷЁйғҪйҷҗеҲ¶дёҖдёӘз«ҷзӮ№жңҖеӨҡдҝқеӯҳ20дёӘCookie
  4. Session еҸҜд»Ҙж”ҫеңЁ ж–Үд»¶гҖҒж•°жҚ®еә“жҲ–еҶ…еӯҳдёӯ
  5. з”ЁжҲ·йӘҢиҜҒиҝҷз§ҚеңәеҗҲдёҖиҲ¬дјҡз”Ё SessionгҖӮеӣ жӯӨпјҢз»ҙжҢҒдёҖдёӘдјҡиҜқзҡ„ж ёеҝғе°ұжҳҜе®ўжҲ·з«Ҝзҡ„е”ҜдёҖж ҮиҜҶпјҢеҚіSession ID
  6. Session зҡ„иҝҗиЎҢдҫқиө–Session IDпјҢиҖҢ Session ID жҳҜеӯҳеңЁ Cookie дёӯзҡ„пјҢд№ҹе°ұжҳҜиҜҙпјҢеҰӮжһңжөҸи§ҲеҷЁзҰҒз”ЁдәҶ CookieпјҢSession д№ҹдјҡеӨұж•ҲпјҲдҪҶжҳҜеҸҜд»ҘйҖҡиҝҮе…¶е®ғж–№ејҸе®һзҺ°пјҢжҜ”еҰӮеңЁ url дёӯдј йҖ’ Session IDпјү
