K8s-CNIзҪ‘з»ңжЁЎеһӢ

иғҢжҷҜ

и·Ёдё»жңәе®№еҷЁй—ҙзҡ„зҪ‘з»ңдә’йҖҡе·Із»ҸжҲҗдёәеҹәжң¬иҰҒжұӮпјҢжӣҙ й«ҳзҡ„иҰҒжұӮеҢ…жӢ¬е®№еҷЁеӣәе®ҡIPең°еқҖгҖҒдёҖдёӘе®№еҷЁеӨҡдёӘIPең°еқҖгҖҒеӨҡдёӘеӯҗзҪ‘йҡ”зҰ»гҖҒ ACLжҺ§еҲ¶зӯ–з•ҘгҖҒдёҺSDNйӣҶжҲҗзӯүгҖӮжүҖд»ҘжҸҗеҮәдәҶContainer Network InterfaceпјҲCNIпјү

CNMзҪ‘з»ңжЁЎеһӢ

дё»иҰҒз»„д»¶еҠҹиғҪпјҡ

  • Network Sandboxпјҡе®№еҷЁеҶ…зҡ„зҪ‘з»ңж ҲпјҢеҢ…жӢ¬зҪ‘з»ңжҺҘеҸЈгҖҒи·Ҝз”ұиЎЁгҖҒDNSзӯүй…ҚзҪ®
  • Endpointпјҡз”ЁдәҺе®№еҷЁеҶ…зҡ„SandboxдёҺеӨ–йғЁзҡ„зҪ‘з»ңзӣёиҝһгҖӮдёҖдёӘEndpointеҸӘиғҪеҠ е…ҘдёҖдёӘNetwork
  • NetworkпјҡеҸҜд»ҘзӣҙжҺҘдә’иҝһзҡ„Endpointзҡ„йӣҶеҗҲгҖӮеҸҜд»ҘйҖҡиҝҮLinuxзҪ‘ жЎҘгҖҒVLANзӯүжҠҖжңҜиҝӣиЎҢе®һзҺ°гҖӮдёҖдёӘNetworkеҢ…еҗ«еӨҡдёӘEndpoint

CNIзҪ‘з»ңжЁЎеһӢ

еңЁCNIжЁЎеһӢдёӯпјҢеҸӘжңүдҝ©дёӘжҰӮеҝөпјҡе®№еҷЁе’ҢзҪ‘з»ңгҖӮ

  • е®№еҷЁпјҡжӢҘжңүзӢ¬з«ӢLinuxзҪ‘з»ңе‘ҪеҗҚз©әй—ҙзҡ„зҺҜеўғ
  • зҪ‘з»ңпјҡдёҖз»„еҸҜд»Ҙдә’иҝһзҡ„дёҖз»„е®һдҪ“пјҢиҝҷдәӣе®һдҪ“жӢҘжңүеҗ„иҮӘзӢ¬з«ӢгҖҒе”Ҝ дёҖзҡ„IPең°еқҖпјҢеҸҜд»ҘжҳҜе®№еҷЁгҖҒзү©зҗҶжңәжҲ–иҖ…е…¶д»–зҪ‘з»ңи®ҫеӨҮпјҲжҜ”еҰӮи·Ҝз”ұеҷЁпјү зӯүгҖӮеҸҜд»Ҙе°Ҷе®№еҷЁж·»еҠ еҲ°дёҖдёӘжҲ–еӨҡдёӘзҪ‘з»ңдёӯпјҢд№ҹеҸҜд»Ҙд»ҺдёҖдёӘжҲ–еӨҡдёӘзҪ‘з»ңдёӯ еҲ йҷӨгҖӮ

еҜ№е®№еҷЁзҪ‘з»ңзҡ„и®ҫзҪ®е’Ңж“ҚдҪңйғҪйҖҡиҝҮжҸ’д»¶пјҲPluginпјүиҝӣиЎҢе…·дҪ“е®һзҺ°пјҢ CNIжҸ’д»¶еҢ…жӢ¬дёӨз§Қзұ»еһӢпјҡCNI Pluginе’ҢIPAMпјҲIP Address Managementпјү PluginгҖӮCNI PluginиҙҹиҙЈдёәе®№еҷЁй…ҚзҪ®зҪ‘з»ңиө„жәҗпјҢIPAM PluginиҙҹиҙЈеҜ№е®№еҷЁ зҡ„IPең°еқҖиҝӣиЎҢеҲҶй…Қе’Ңз®ЎзҗҶгҖӮIPAM PluginдҪңдёәCNI Pluginзҡ„дёҖйғЁеҲҶпјҢдёҺ CNI PluginдёҖиө·е·ҘдҪң

е®№еҷЁиҝҗиЎҢж—¶е’ҢCNIжҸ’д»¶зҡ„е…ізі»е’Ңе·ҘдҪңжңәеҲ¶

е°Ҷе®№еҷЁж·»еҠ еҲ°зҪ‘з»ңдёӯжҲ–иҖ…еҲ йҷӨжҹҗдёӘзҪ‘з»ңжҳҜз”ұе®№еҷЁиҝҗиЎҢж—¶пјҲruntimeпјү е’ҢCNIжҸ’д»¶е®ҢжҲҗзҡ„гҖӮдёӢйқўжҳҜдёҖдәӣе·ҘдҪңеҺҹеҲҷпјҡ

  1. е®№еҷЁиҝҗиЎҢж—¶еҝ…йЎ»еңЁи°ғз”Ёд»»ж„ҸжҸ’д»¶еүҚдёәе®№еҷЁеҲӣе»әдёҖдёӘж–°зҡ„зҪ‘з»ңе‘ҪеҗҚз©әй—ҙ
  2. е®№еҷЁиҝҗиЎҢж—¶еҝ…йЎ»зЎ®е®ҡжӯӨе®№еҷЁеҪ’еұһзҪ‘з»ңпјҲе…Ғи®ёеӨҡдёӘпјүпјҢд»ҘеҸҠжҜҸдёӘзҪ‘з»ңеҝ…йЎ»жү§иЎҢе“ӘдёӘжҸ’д»¶
  3. зҪ‘з»ңй…ҚзҪ®дёәJSONж јејҸпјҢдҫҝдәҺж–Үд»¶еӯҳеӮЁгҖӮзҪ‘з»ңй…ҚзҪ®еҢ…жӢ¬еҝ… еЎ«еӯ—ж®өпјҢдҫӢеҰӮnameе’ҢtypeпјҢд»ҘеҸҠжҸ’д»¶пјҲзұ»еһӢпјүзү№жңүзҡ„еӯ—ж®өгҖӮзҪ‘з»ңй…ҚзҪ®е…Ғ и®ёеңЁи°ғз”Ёж—¶жӣҙж”№еӯ—ж®өзҡ„еҖјгҖӮдёәжӯӨпјҢеҝ…йЎ»еңЁеҸҜйҖүеӯ—ж®өargsдёӯеҢ…еҗ«йңҖиҰҒеҸҳжӣҙ зҡ„дҝЎжҒҜ
  4. е®№еҷЁиҝҗиЎҢж—¶еҝ…йЎ»жҢүз…§е…ҲеҗҺйЎәеәҸдёәжҜҸдёӘзҪ‘з»ңиҝҗиЎҢжҸ’д»¶е°Ҷе®№еҷЁж·»еҠ  еҲ°жҜҸдёӘзҪ‘з»ңдёӯ
  5. еҷЁз”ҹе‘Ҫе‘Ёжңҹз»“жқҹеҗҺпјҢе®№еҷЁиҝҗиЎҢж—¶еҝ…йЎ»д»ҘеҸҚеҗ‘йЎәеәҸпјҲзӣёеҜ№дәҺж·» еҠ е®№еҷЁжү§иЎҢйЎәеәҸпјүжү§иЎҢжҸ’д»¶пјҢд»ҘдҪҝе®№еҷЁдёҺзҪ‘з»ңж–ӯејҖиҝһжҺҘ
  6. е®№еҷЁиҝҗиЎҢж—¶дёҖе®ҡдёҚиғҪдёәеҗҢдёҖдёӘе®№еҷЁзҡ„и°ғз”Ёжү§иЎҢе№¶иЎҢпјҲparallelпјү ж“ҚдҪңпјҢдҪҶеҸҜд»ҘдёәеӨҡдёӘдёҚеҗҢе®№еҷЁзҡ„и°ғз”Ёжү§иЎҢе№¶иЎҢж“ҚдҪң
  7. е®№еҷЁиҝҗиЎҢж—¶еҝ…йЎ»еҜ№е®№еҷЁзҡ„ADDе’ҢDELж“ҚдҪңи®ҫзҪ®йЎәеәҸпјҢд»ҘдҪҝеҫ— ADDж“ҚдҪңжңҖз»Ҳи·ҹйҡҸзӣёеә”зҡ„DELж“ҚдҪңгҖӮDELж“ҚдҪңеҗҺйқўеҸҜиғҪдјҡжңүе…¶д»–DELж“Қ дҪңпјҢдҪҶжҸ’д»¶еә”иҮӘз”ұеӨ„зҗҶеӨҡдёӘDELж“ҚдҪңпјҲеҚіеӨҡдёӘDELж“ҚдҪңеә”иҜҘжҳҜе№Ӯзӯү зҡ„пјү
  8. е®№еҷЁеҝ…йЎ»з”ұContainerIDиҝӣиЎҢе”ҜдёҖж ҮиҜҶгҖӮеӯҳеӮЁзҠ¶жҖҒзҡ„жҸ’д»¶еә”дҪҝз”Ё иҒ”еҗҲдё»й”®пјҲnetwork nameгҖҒCNI_CONTAINERIDгҖҒCNI_IFNAMEпјүиҝӣиЎҢеӯҳеӮЁ
  9. е®№еҷЁиҝҗиЎҢж—¶дёҚеҫ—дёәеҗҢдёҖдёӘе®һдҫӢпјҲз”ұиҒ”еҗҲдё»й”®network nameгҖҒ CNI_CONTAINERIDгҖҒCNI_IFNAMEиҝӣиЎҢж ҮиҜҶпјүи°ғз”ЁдёӨж¬ЎADDж“ҚдҪң пјҲж— зӣёеә”зҡ„DELж“ҚдҪңпјүгҖӮеҜ№еҗҢдёҖдёӘе®№еҷЁпјҲContainerIDпјүпјҢд»…еңЁжҜҸж¬ЎADD ж“ҚдҪңйғҪдҪҝз”ЁдёҚеҗҢзҡ„зҪ‘з»ңжҺҘеҸЈеҗҚз§°ж—¶пјҢжүҚеҸҜд»ҘеӨҡж¬Ўж·»еҠ еҲ°зү№е®ҡзҡ„зҪ‘з»ңдёӯ
  10. йҷӨйқһжҳҺзЎ®ж Үи®°дёәеҸҜйҖүй…ҚзҪ®пјҢCNIз»“жһ„дёӯзҡ„еӯ—ж®өпјҲдҫӢеҰӮNetwork Configurationе’ҢCNI Plugin ResultпјүйғҪжҳҜеҝ…еЎ«еӯ—ж®ө

CNI Plugin

CNI Pluginеҝ…йЎ»жҳҜдёҖдёӘеҸҜжү§иЎҢзЁӢеәҸпјҢз”ұе®№еҷЁз®ЎзҗҶзі»з»ҹпјҲеҰӮ Kubernetesпјүи°ғз”ЁгҖӮ

дё»иҰҒеҠҹиғҪ

иҙҹиҙЈжҠҠзҪ‘з»ңжҺҘеҸЈжҸ’е…Ҙе®№еҷЁзҪ‘з»ңеҗҚз§°з©әй—ҙпјҢе№¶еңЁдё»жңәдёҠжү§иЎҢеҝ…иҰҒжӣҙж”№пјҢз„¶еҗҺи°ғз”ЁйҖӮеҪ“зҡ„IPAMжҸ’д»¶пјҢжҠҠIPеҲҶй…Қз»ҷзҪ‘з»ңжҺҘеҸЈпјҢе№¶и®ҫзҪ®жӯЈзЎ®зҡ„и·Ҝз”ұи§„еҲҷгҖӮ

CNI PluginйңҖиҰҒж”ҜжҢҒзҡ„ж“ҚдҪңеҢ…жӢ¬ADDпјҲж·»еҠ пјүгҖҒDELETEпјҲеҲ  йҷӨпјүгҖҒCHECKпјҲжЈҖжҹҘпјүе’ҢVERSIONпјҲзүҲжң¬жҹҘиҜўпјүгҖӮиҝҷдәӣж“ҚдҪңзҡ„е…·дҪ“е®һ зҺ°еқҮз”ұCNI PluginеҸҜжү§иЎҢзЁӢеәҸе®ҢжҲҗгҖӮ

ADD

дё»иҰҒеҠҹиғҪ

ADDпјҡе°Ҷе®№еҷЁж·»еҠ еҲ°жҹҗдёӘзҪ‘з»ңдёӯпјҢдё»иҰҒиҝҮзЁӢдёәеңЁContainer RuntimeеҲӣе»әе®№еҷЁж—¶пјҢе…ҲеҲӣе»әеҘҪе®№еҷЁеҶ…зҡ„зҪ‘з»ңе‘ҪеҗҚз©әй—ҙпјҢз„¶еҗҺи°ғз”ЁCNIжҸ’ д»¶дёәиҜҘnetnsе®ҢжҲҗе®№еҷЁзҪ‘з»ңзҡ„й…ҚзҪ®гҖӮ

ADDж“ҚдҪңзҡ„еҸӮж•°еҰӮдёӢгҖӮ

  • Container IDпјҡе®№еҷЁIDпјҢдёәе®№еҷЁзҡ„е”ҜдёҖж ҮиҜҶгҖӮ

  • Network namespace pathпјҡе®№еҷЁзҡ„зҪ‘з»ңе‘ҪеҗҚз©әй—ҙи·Ҝеҫ„пјҢдҫӢ еҰӮ/proc/[pid]/ns/netгҖӮ

  • Network configurationпјҡзҪ‘з»ңй…ҚзҪ®JSONж–ҮжЎЈпјҢз”ЁдәҺжҸҸиҝ°е®№еҷЁеҫ… еҠ е…Ҙзҡ„зҪ‘з»ңгҖӮ

  • Extra argumentsпјҡе…¶д»–еҸӮж•°пјҢжҸҗдҫӣдәҶеҸҰдёҖз§Қд»ҘжҜҸдёӘе®№еҷЁдёәеҹәзЎҖ зҡ„CNIжҸ’д»¶з®ҖеҚ•й…ҚзҪ®жңәеҲ¶гҖӮ

  • Name of the interface inside the containerпјҡе®№еҷЁеҶ…зҡ„иҷҡжӢҹзҪ‘еҚЎеҗҚз§°

ADDж“ҚдҪңзҡ„з»“жһңдҝЎжҒҜеҢ…еҗ«д»ҘдёӢеҸӮж•°

  • Interfaces listпјҡзҪ‘з»ңжҺҘеҸЈеҲ—иЎЁпјҢж №жҚ®Pluginзҡ„е®һзҺ°пјҢеҸҜиғҪеҢ…жӢ¬ Sandbox InterfaceеҗҚз§°гҖҒдё»жңәInterfaceеҗҚз§°гҖҒжҜҸдёӘInterfaceзҡ„ең°еқҖзӯүдҝЎ жҒҜгҖӮ
  • Interfaces listпјҡзҪ‘з»ңжҺҘеҸЈеҲ—иЎЁпјҢж №жҚ®Pluginзҡ„е®һзҺ°пјҢеҸҜиғҪеҢ…жӢ¬ Sandbox InterfaceеҗҚз§°гҖҒдё»жңәInterfaceеҗҚз§°гҖҒжҜҸдёӘInterfaceзҡ„ең°еқҖзӯүдҝЎ жҒҜгҖӮ
  • IP configuration assigned to each interfaceпјҡеҲҶй…Қз»ҷжҜҸдёӘзҪ‘з»ңжҺҘеҸЈ зҡ„IPv4жҲ–IPv6ең°еқҖгҖҒзҪ‘е…іең°еқҖе’Ңи·Ҝз”ұдҝЎжҒҜгҖӮ
  • DNS informationпјҡDNSзӣёе…ідҝЎжҒҜпјҢеҢ…жӢ¬еҹҹеҗҚжңҚеҠЎеҷЁпјҲname serverпјүгҖҒеҹҹеҗҚдҝЎжҒҜпјҲdomainпјүгҖҒжҗңзҙўеҗҺзјҖпјҲsearch domainsпјүгҖҒDNSйҖү йЎ№пјҲoptionsпјү

DEL

дё»иҰҒеҠҹиғҪ

DELпјҡеңЁе®№еҷЁй”ҖжҜҒж—¶е°Ҷе®№еҷЁд»ҺжҹҗдёӘзҪ‘з»ңдёӯеҲ йҷӨгҖӮ

DELж“ҚдҪңзҡ„ еҸӮж•°еҰӮдёӢ

  • Container IDпјҡе®№еҷЁID

  • Network namespace pathпјҡе®№еҷЁзҡ„зҪ‘з»ңе‘ҪеҗҚз©әй—ҙи·Ҝеҫ„пјҢдҫӢ еҰӮ/proc/[pid]/ns/netгҖӮ

  • Network configurationпјҡзҪ‘з»ңй…ҚзҪ®JSONж–ҮжЎЈпјҢз”ЁдәҺжҸҸиҝ°е®№еҷЁеҫ… еҠ е…Ҙзҡ„зҪ‘з»ңгҖӮ

  • Extra argumentsпјҡе…¶д»–еҸӮж•°гҖӮ

  • Name of the interface inside the containerпјҡе®№еҷЁеҶ…зҡ„зҪ‘еҚЎеҗҚгҖӮ

жү§иЎҢDELж“ҚдҪңж—¶йңҖиҰҒжіЁж„ҸеҰӮдёӢдәӢйЎ№

  • жүҖжңүеҸӮж•°йғҪеҝ…йЎ»дёҺжү§иЎҢADDж“ҚдҪңж—¶зӣёеҗҢгҖӮ
  • DELж“ҚдҪңеә”иҜҘйҮҠж”ҫе®№еҷЁпјҲContainerIDпјүеҚ з”Ёзҡ„жүҖжңүзҪ‘з»ңиө„жәҗгҖӮ
  • еҰӮжһңеүҚдёҖдёӘж“ҚдҪңжҳҜADDпјҢеҲҷеә”еңЁзҪ‘з»ңжҸ’д»¶зҡ„й…ҚзҪ®ж–Үд»¶JSONдёӯ иЎҘе……prevResultеӯ—ж®өпјҢд»Ҙж ҮжҳҺеүҚдёҖдёӘж“ҚдҪңзҡ„з»“жһңпјҲе®№еҷЁиҝҗиЎҢж—¶еҸҜиғҪдјҡзј“ еӯҳADDзҡ„з»“жһңпјүгҖӮ
