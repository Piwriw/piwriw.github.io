K8s 公网部署集群

BackGroud

在公网环境下，或者需要跨局域网组成集群下。需要让Master以公网IP启动，而kubeadm init 参数启动，只支持局域网。

Deploy

  1. 获取配置文件kubead config default > k8s-init.yaml，localAPIEndpoint.advertiseAddress就是公网IP地址

最新使用 kubeadm config print init-defaults > k8s-init.yaml

  1. 启动集群
    kubeadm init --config k8s-init.yaml

  2. mkdir -p $HOME/.kube
sudo cp -i /etc/kubernetes/admin.conf $HOME/.kube/config
sudo chown $(id -u):$(id -g) $HOME/.kube/config

  3. 查看是否正确，cat /root/.kube/config,的IP和配置文件。
