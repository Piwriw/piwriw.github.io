IngressжҺ§еҲ¶еҷЁ

IngressжҺ§еҲ¶еҷЁдёҚйҡҸзқҖK8sеҗҜеҠЁе®үиЈ…

K8sе®ҳзҪ‘ж”ҜжҢҒз»ҙжҠӨзҡ„IngressжҺ§еҲ¶еҷЁ

  • Kubernetes дҪңдёәдёҖдёӘйЎ№зӣ®пјҢзӣ®еүҚж”ҜжҢҒе’Ңз»ҙжҠӨ AWSгҖҒ GCE е’Ң Nginx Ingress жҺ§еҲ¶еҷЁ

з¬¬дёүж–№Ingressпјҡ

  • AKS еә”з”ЁзЁӢеәҸзҪ‘е…і Ingress жҺ§еҲ¶еҷЁ жҳҜдёҖдёӘй…ҚзҪ® Azure еә”з”ЁзЁӢеәҸзҪ‘е…і зҡ„ Ingress жҺ§еҲ¶еҷЁгҖӮ

  • йҳҝйҮҢдә‘ MSE Ingress жҳҜдёҖдёӘ Ingress жҺ§еҲ¶еҷЁпјҢе®ғиҙҹиҙЈй…ҚзҪ®йҳҝйҮҢдә‘еҺҹз”ҹзҪ‘е…іпјҢ д№ҹжҳҜ Higress зҡ„е•ҶдёҡзүҲжң¬гҖӮ

  • Apache APISIX Ingress жҺ§еҲ¶еҷЁ жҳҜдёҖдёӘеҹәдәҺ Apache APISIX зҪ‘е…і зҡ„ Ingress жҺ§еҲ¶еҷЁгҖӮ

  • Avi Kubernetes Operator дҪҝз”Ё VMware NSX Advanced Load Balancer жҸҗдҫӣз¬¬ 4 еҲ°з¬¬ 7 еұӮзҡ„иҙҹиҪҪеқҮиЎЎгҖӮ

  • BFE Ingress жҺ§еҲ¶еҷЁжҳҜдёҖдёӘеҹәдәҺ BFE зҡ„ Ingress жҺ§еҲ¶еҷЁгҖӮ

  • Cilium Ingress жҺ§еҲ¶еҷЁжҳҜдёҖдёӘз”ұ Cilium еҮәе“Ғж”ҜжҢҒзҡ„ Ingress жҺ§еҲ¶еҷЁгҖӮ

  • Citrix Ingress жҺ§еҲ¶еҷЁ еҸҜд»Ҙз”ЁжқҘдёҺ Citrix Application Delivery Controller дёҖиө·дҪҝз”ЁгҖӮ

  • Contour жҳҜдёҖдёӘеҹәдәҺ Envoy зҡ„ Ingress жҺ§еҲ¶еҷЁгҖӮ

  • Emissary-Ingress API зҪ‘е…іжҳҜдёҖдёӘеҹәдәҺ Envoy зҡ„е…ҘеҸЈжҺ§еҲ¶еҷЁгҖӮ

  • EnRoute жҳҜдёҖдёӘеҹәдәҺ Envoy зҡ„ API зҪ‘е…іпјҢеҸҜд»Ҙз”ЁдҪң Ingress жҺ§еҲ¶еҷЁгҖӮ

  • Easegress IngressController жҳҜдёҖдёӘеҹәдәҺ Easegress зҡ„ API зҪ‘е…іпјҢеҸҜд»Ҙз”ЁдҪң Ingress жҺ§еҲ¶еҷЁгҖӮ

  • F5 BIG-IP зҡ„ з”ЁдәҺ Kubernetes зҡ„е®№еҷЁ Ingress жңҚеҠЎ и®©дҪ иғҪеӨҹдҪҝз”Ё Ingress жқҘй…ҚзҪ® F5 BIG-IP иҷҡжӢҹжңҚеҠЎеҷЁгҖӮ

  • FortiADC Ingress жҺ§еҲ¶еҷЁ ж”ҜжҢҒ Kubernetes Ingress иө„жәҗпјҢе№¶е…Ғи®ёдҪ д»Һ Kubernetes з®ЎзҗҶ FortiADC еҜ№иұЎгҖӮ

  • Gloo жҳҜдёҖдёӘејҖжәҗзҡ„гҖҒеҹәдәҺ Envoy зҡ„ Ingress жҺ§еҲ¶еҷЁпјҢиғҪеӨҹжҸҗдҫӣ API зҪ‘е…іеҠҹиғҪгҖӮ

  • HAProxy Ingress жҳҜдёҖдёӘй’ҲеҜ№ HAProxy зҡ„ Ingress жҺ§еҲ¶еҷЁгҖӮ

  • Higress жҳҜдёҖдёӘеҹәдәҺ Envoy зҡ„ API зҪ‘е…іпјҢ еҸҜд»ҘдҪңдёәдёҖдёӘ Ingress жҺ§еҲ¶еҷЁиҝҗиЎҢгҖӮ

  • з”ЁдәҺ Kubernetes зҡ„ HAProxy Ingress жҺ§еҲ¶еҷЁ д№ҹжҳҜдёҖдёӘй’ҲеҜ№ HAProxy зҡ„ Ingress жҺ§еҲ¶еҷЁгҖӮ

  • Istio Ingress жҳҜдёҖдёӘеҹәдәҺ Istio зҡ„ Ingress жҺ§еҲ¶еҷЁгҖӮ

  • з”ЁдәҺ Kubernetes зҡ„ Kong Ingress жҺ§еҲ¶еҷЁ жҳҜдёҖдёӘз”ЁжқҘй©ұеҠЁ Kong Gateway зҡ„ Ingress жҺ§еҲ¶еҷЁгҖӮ

  • Kusk Gateway жҳҜдёҖдёӘеҹәдәҺ Envoy зҡ„гҖҒ OpenAPI й©ұеҠЁзҡ„ Ingress жҺ§еҲ¶еҷЁгҖӮ

  • з”ЁдәҺ Kubernetes зҡ„ NGINX Ingress жҺ§еҲ¶еҷЁ иғҪеӨҹдёҺ NGINX зҪ‘йЎөжңҚеҠЎеҷЁпјҲдҪңдёәд»ЈзҗҶпјүдёҖиө·дҪҝз”ЁгҖӮ

  • ngrok Kubernetes Ingress жҺ§еҲ¶еҷЁ жҳҜдёҖдёӘејҖжәҗжҺ§еҲ¶еҷЁпјҢйҖҡиҝҮдҪҝз”Ё ngrok е№іеҸ°дёәдҪ зҡ„ K8s жңҚеҠЎж·»еҠ е®үе…Ёзҡ„е…¬ејҖи®ҝй—®жқғйҷҗгҖӮ

  • OCI Native Ingress Controller жҳҜдёҖдёӘйҖӮз”ЁдәҺ Oracle Cloud Infrastructure зҡ„ Ingress жҺ§еҲ¶еҷЁпјҢеҸҜеё®еҠ©дҪ з®ЎзҗҶ OCI иҙҹиҪҪеқҮиЎЎгҖӮ

  • OpenNJet Ingress Controller жҳҜдёҖдёӘеҹәдәҺ OpenNJet зҡ„ Ingress жҺ§еҲ¶еҷЁгҖӮ

  • Pomerium Ingress жҺ§еҲ¶еҷЁ еҹәдәҺ PomeriumпјҢиғҪжҸҗдҫӣдёҠдёӢж–Үж„ҹзҹҘзҡ„еҮҶе…Ҙзӯ–з•ҘгҖӮ

  • Skipper HTTP и·Ҝз”ұеҷЁе’ҢеҸҚеҗ‘д»ЈзҗҶеҸҜз”ЁдәҺжңҚеҠЎз»„иЈ…пјҢж”ҜжҢҒеҢ…жӢ¬ Kubernetes Ingress иҝҷзұ»дҪҝз”ЁеңәжҷҜпјҢжҳҜдёҖдёӘз”Ёд»Ҙжһ„йҖ дҪ иҮӘе·ұзҡ„е®ҡеҲ¶д»ЈзҗҶзҡ„еә“гҖӮ

  • Traefik Kubernetes Ingress жҸҗдҫӣзЁӢеәҸ жҳҜдёҖдёӘз”ЁдәҺ Traefik д»ЈзҗҶзҡ„ Ingress жҺ§еҲ¶еҷЁгҖӮ

  • Tyk Operator дҪҝз”ЁиҮӘе®ҡд№үиө„жәҗжү©еұ• IngressпјҢдёәд№ӢеёҰжқҘ API з®ЎзҗҶиғҪеҠӣгҖӮTyk Operator дҪҝз”ЁејҖжәҗзҡ„ Tyk Gateway & Tyk Cloud жҺ§еҲ¶йқўгҖӮ

  • Voyager жҳҜдёҖдёӘй’ҲеҜ№ HAProxy зҡ„ Ingress жҺ§еҲ¶еҷЁгҖӮ

  • Wallarm Ingress Controller жҳҜжҸҗдҫӣ WAAPпјҲWAFпјү е’Ң API е®үе…ЁеҠҹиғҪзҡ„ Ingress ControllerгҖӮ
