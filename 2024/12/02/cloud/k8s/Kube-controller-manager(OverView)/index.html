kube-controller-managerиҝҗиЎҢдәҶеӨҡдёӘжҺ§еҲ¶еҷЁгҖӮжҺ§еҲ¶еҷЁйҖҡиҝҮInformerжңәеҲ¶зӣ‘еҗ¬иө„жәҗеҜ№иұЎзҡ„AddгҖҒUpdateгҖҒDeleteдәӢд»¶

    жҺ§еҲ¶еҷЁйҖҡиҝҮK8sзҡ„InformerгҖҒWatchжңәеҲ¶пјҢзӣ‘еҗ¬йӣҶзҫӨдёӯиө„жәҗеҜ№иұЎзҡ„AddгҖҒUpdateгҖҒDeleteдәӢд»¶пјҢдёәдёҚеҗҢзҡ„дәӢд»¶жіЁеҶҢдёҚеҗҢзҡ„еӣһи°ғеҮҪж•°пјҢе№¶дё”е°Ҷзӣ‘еҗ¬зҡ„иө„жәҗзј“еӯҳеҜ№иұЎеңЁInformerдёӯ

  2. е°Ҷиө„жәҗеҜ№иұЎзҡ„KeyеҠ е…ҘеҲ°е·ҘдҪңйҳҹеҲ—дёӯ

  4. WorkerеҚҸзЁӢж №жҚ®KeyиҺ·еҸ–е®Ңж•ҙзҡ„иө„жәҗеҜ№иұЎ

    WorkerеҚҸзЁӢд»Һе·ҘдҪңйҳҹеҲ—дёӯиҺ·еҸ–KeyеҗҺпјҢж №жҚ®Keyд»ҺInformerзј“еӯҳдёӯиҺ·еҸ–е®Ңж•ҙзҡ„иө„жәҗеҜ№иұЎгҖӮInformerдёҖзӣҙеңЁзӣ‘еҗ¬иө„жәҗеҜ№иұЎзҡ„еҸҳеҢ–пјҢеӣ жӯӨзј“еӯҳзҡ„жҖ»жҳҜжңҖж–°зҡ„иө„жәҗеҜ№иұЎзүҲжң¬гҖӮжҺ§еҲ¶еҷЁе°ҶKeyиҖҢйқһиө„жәҗеҜ№иұЎжң¬иә«еҠ е…Ҙе·ҘдҪңйҳҹеҲ—пјҢеҸҜд»ҘдҪҝз”ЁWorkerеҚҸзЁӢжҖ»жҳҜд»ҘжңҖж–°зҡ„иө„жәҗзүҲжң¬д»ҘеҮҶиҝӣи°ғи°җ

  5. WorkerеҚҸзЁӢжү§иЎҢиө„жәҗеҜ№иұЎReconcileи°ғеҚҸ

    WorkerеҚҸзЁӢеңЁиҺ·еҸ–е®Ңж•ҙзҡ„иө„жәҗеҜ№иұЎеҗҺпјҢејҖе§Ӣжү§иЎҢиө„жәҗеҜ№иұЎReconcileи°ғи°җ

    еҰӮжһңжІЎжңүеҲ°иҫҫеҲ°жңҹжңӣзҠ¶жҖҒпјҢжҲ–иҖ…еҗҢжӯҘеҮәзҺ°й—®йўҳпјҢе°ұдјҡйҮҚж–°ж”ҫеӣһеҲ°е·ҘдҪңйҳҹеҲ—дёӯ

  • жҺ§еҲ¶еҷЁдјҡеҗҜеҠЁеӨҡдёӘWorkerеҚҸзЁӢпјҢдҪҶжҳҜе…ұдә«дёҖдёӘе·ҘдҪңйҳҹеҲ—пјҢе·ҘдҪңйҳҹеҲ—еҗ«жңүеҺ»йҮҚпјҢдҝқиҜҒдәҶеҗҢдёҖдёӘиө„жәҗеҜ№иұЎдёҚдјҡеҗҢж—¶иў«еӨҡдёӘWorkerеӨ„зҗҶ
