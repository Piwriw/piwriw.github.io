1.2.3 VXLAN 原理详解

VXLAN (Virtual Extensible LAN) 是一种网络虚拟化技术，通过 UDP 封装在三层网络上创建二层虚拟网络。

VXLAN 核心概念
VXLAN 数据包封装原理
VXLAN 封装/解封装设备 (VTEP) 工作机制
VXLAN 与 VLAN 对比
特性 VXLAN VLAN
网络标识 24 位 VNI (16M+ 网络) 12 位 VLAN ID (4K 网络)
二层范围 可跨三层网络 局限于单个交换机
封装方式 UDP 封装 无封装 (纯二层)
隧道 需要 VTEP 不需要
规模 支持 16M+ 虚拟网络 4K 虚拟网络
典型应用 云环境、多租户 数据中心网络隔离
